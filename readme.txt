- 기존의 얼굴인식 시스템을 이미 데이터베이스가 확보된 범죄자를 검출하는 부분에 접목한 프로그램입니다.


* 웹캠 구동 관련해서 메모리 관리가 잘 되지 않아 PCA 인식을 몇 차례 하다 보면
  메모리 점유율이 점점 올라가고 버튼이 잘 반응하지 않는 버그가 있습니다.

  프로그램 사용법은 다음과 같습니다.

- HMM을 먼저 구동시킵니다. 이 때 실행파일 경로는 Debug 폴더가 되면 안됩니다.(프로젝트 폴더여야 함)
  HMM과 PCA의 OpenCV 라이브러리 버전이 호환이 되지 않아 소스 코드 레벨의 알고리즘 결합이
  어려워 프로그램 간 유기적인 결합을 통해 문제를 해결하였습니다.

- PCA를 구동시켜 얼굴 인식을 수행합니다.(1차 인식)
  먼저 '얼굴탐지' 버튼으로 웹캠으로 촬영된 사진으로부터 얼굴 영역을 추출합니다.
  얼굴 영역이 추출되면 '인식' 버튼으로 기존에 있던 사진 데이터베이스들과 비교합니다.


- 1차 인식 과정(PCA)에서 임계값 내의 유사 데이터가 없는 경우
  Background로 구동되고 있는 HMM 알고리즘이 2차 인식 과정을 수행합니다.
  만약 2차 인식 과정(HMM)에서도 임계값 내의 유사 데이터가 없는 경우
  저장된 데이터베이스에는 유사한 인물이 없는 것으로 간주하고 인식을 마칩니다.
